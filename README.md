# footMe

The new footballers' portal.

## Stack iniziale

- Mobile: React Native con Expo e TypeScript
- Backend: Supabase
- Repository structure: npm workspaces con app mobile in `apps/mobile`

## Quickstart

Prerequisiti:

- Node.js 20+
- npm 10+

Bootstrap locale:

```bash
npm install
npm run dev:mobile
```

Per aprire subito il QR code in Expo Go e condividere i progressi anche fuori dalla rete locale:

```bash
npm run qr:mobile
```

Il comando avvia Expo in modalità tunnel e mostra il QR code nel terminale, pronto da scansionare con Expo Go.
